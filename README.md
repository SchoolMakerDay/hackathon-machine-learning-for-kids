# hackathon-machine-learning-for-kids
Materiale di preparazione per l'hackathon SMD 2026 - Secondaria di I grado 

Questo repository propone alcuni esempi di sviluppo di progetti sulla piattaforma "Machine Learning for Kids".<br>
La piattaforma consente lo sviluppo di giochi realizzati con una versione modificata di Scratch dotata di un estensione di addestramento deò computer al riconoscimento mediante intelligenza artificiale.<br>
Come nel Machine Learning professionale gli sviluppatori devono predisporre dei modelli di riconoscimento di oggetti tramite "etichette" a cui vanno associati esempi. Le etichette possono riguardare il riconoscimento di immagini, suoni, testo, numeri.<br>
Dopo l'addestramento il gioco ricevendo altri oggetti riconducibili alle etichette su cui è stato addestrato riconosce gli oggetti.
La piattaforma è accessibile all'indirizzo:<br>
https://machinelearningforkids.co.uk/<br>
L'ambiente di sviluppo Scratch è disponbile all'indirizzo:<br>
https://machinelearningforkids.co.uk/scratch/<br>
Gli esempi di sviluppo con documentazione e risorse sono disponibili nelle sottocartelle di questo repository
